# 🛍️ Style Hub

A modern and responsive **Fashion E-commerce Landing Page** built using **HTML, CSS, and JavaScript**. Style Hub features a clean UI, stylish typography, smooth animations, and a fully responsive design suitable for desktop, tablet, and mobile devices.

## 🚀 Getting Started


### Open the Project

```bash

```

Simply open **index.html** in your browser.
